# 엔딩 분기 설계

## 전제

당신은 대한민국의 평범한 가정에서 태어나 평범한 삶을 살아가게 된다.
14일 이후 당신의 인생이 결정된다.

---

## 엔딩 직업군 분류

엔딩은 "직업/역할"로 분류. 같은 직업군 안에서도 스텟 조합에 따라 세부 결말이 달라짐.

### 바닥 (돈 0 or 극빈)

| 직업 | 조건 | 설명 |
|------|------|------|
| 노숙자 | 돈 0 + 특별 이벤트 없음 | 배금도시의 바닥. 아무것도 남지 않았다. |
| 백수 | 돈 낮음 + 아무 루트도 안 탐 | 14일을 그냥 흘려보냈다. |

### 일반직

| 직업 | 조건 | 설명 |
|------|------|------|
| 일용직 노동자 | 특별 조건 없음 | 하루하루 버텼다. 특별한 일은 없었다. |
| 회사원 | 평판 보통 + 범죄도 낮음 | 성실하게 버텼다. 이름 없이 살아간다. |
| 공무원 | 지능 높음 + 평판 높음 + 범죄도 낮음 | 안정을 택했다. 배금도시에서 드문 선택. |

### 전문직

| 직업 | 조건 | 설명 |
|------|------|------|
| 의사 / 변호사 | 지능 최고 + 범죄도 낮음 + 전문직 이벤트 클리어 | 오래 걸렸지만 단단한 자리를 잡았다. |
| 유튜버 / 인플루언서 | 외모 높음 + 평판 높음 + 휴대폰 적극 활용 | 얼굴로 먹고산다. 언제 끝날지 모른다. |
| 예술가 | 지능 높음 + 돈 낮음 + 창작 이벤트 클리어 | 배금도시에서 돈을 못 벌었지만 뭔가를 남겼다. |
| 유명 예술가 | 지능 최고 + 평판 높음 + 창작 이벤트 전부 클리어 | 뒤늦게 세상이 알아봤다. 돈은 나중에 따라왔다. |

### 사업가

| 직업 | 조건 | 설명 |
|------|------|------|
| 자영업자 | 돈 높음 + 창업 이벤트 클리어 | 작은 가게 하나 차렸다. |
| 주식 트레이더 | 지능 높음 + 투자 이벤트 클리어 | 숫자로 먹고산다. |
| 도박사 | 도박 이벤트 다수 + 운 조건 | 전부 잃거나 전부 따거나. 중간이 없다. |
| 투자자 | 지능 높음 + 돈 높음 + 투자 이벤트 클리어 | 돈이 돈을 번다. |
| 대기업 오너 | 지능 최고 + 돈 최고 + 평판 높음 | 배금도시를 손에 쥐었다. |

### 권력자

| 직업 | 조건 | 설명 |
|------|------|------|
| 정치인 | 평판 높음 + 지능 높음 + 범죄도 낮음 | 민심을 얻었다. |
| 대한민국 대통령 | 지능 최고 + 평판 최고 + 범죄도 낮음 + 정치 이벤트 전부 클리어 | 흙수저의 끝. 극히 드문 엔딩. |

### 범죄자

| 직업 | 조건 | 설명 |
|------|------|------|
| 소매치기 | 범죄도 10 이상 | 작은 범죄로 하루를 버텼다. |
| 조폭 | 범죄도 25 이상 + 폭력 이벤트 다수 | 주먹으로 먹고산다. |
| 사기꾼 | 범죄도 25 이상 + 사기 이벤트 | 말로 먹고산다. |
| 마약상 | 범죄도 40 이상 + 마약 이벤트 해금 | 빠른 돈. 더 빠른 추락. |
| 보이스피싱 조직장 | 범죄도 50 이상 + 보이스피싱 루트 완주 | 피해자의 가족이었던 주인공이 조직을 이끈다. |
| 살인마 | 범죄도 90 이상 + 살인 이벤트 다수 | 뉴스에 이름이 오르내린다. |

### 어둠의 권력자

| 직업 | 조건 | 설명 |
|------|------|------|
| 사이비 교주 | 범죄도 70 이상 + 사이비 루트 완주 | 처음엔 당할 뻔했던 그 수법을 직접 쓴다. |
| 쿠데타 주동자 | 범죄도 90 이상 + 지능 낮음 | 무력으로 잡은 권력. 오래가지 못한다. |
| 어둠의 권력자 | 범죄도 90 이상 + 지능 최고 + 정치 쿠데타 해금 | 합법과 불법의 경계. 대한민국을 쥐고 흔든다. |

### 히든 엔딩

| 직업 | 조건 | 설명 |
|------|------|------|
| 배금도시 탈출 | 조건 미공개 (공략 없이 발견하기 어렵게) | 돈이 전부인 세상을 거부하고 떠났다. |

## 최종 랭킹 화면

14일이 끝나면 랭킹을 보여줌.
내 결과만 보여주는 게 아니라 다른 플레이어들이 걸어온 삶과 비교.
탭으로 기준을 바꿔서 여러 랭킹을 볼 수 있음.

### 랭킹 탭 구성

```
[ 재산 ]  [ 범죄력 ]  [ 평판 ]  [ 지능 ]
```

각 탭마다 기준이 다른 랭킹이 펼쳐짐.
같은 플레이어가 재산 랭킹 1위여도 평판 랭킹에선 꼴찌일 수 있음.

```
─────────────────────────────────────
  배금도시 재산 랭킹
─────────────────────────────────────
  1위   어둠의 권력자   3조 2,000억 원
  2위   대기업 오너     890억 원
  3위   사이비 교주     12억 4,000만 원
  4위   투자자          3억 8,000만 원
  5위   대통령          2억 1,000만 원
  ...
  9위   나 (사기꾼)     4,200만 원
  ...
  18위  자영업자        890만 원
  ...
  31위  일용직 노동자   240만 원
─────────────────────────────────────
  배금도시에서 돈이 전부다.
  당신의 삶은 몇 위입니까?
─────────────────────────────────────
```

### 현실 기준 금액 참고

게임 내 돈 스케일 기준표. 코드 구현 시 이 범위에 맞게 payMin/payMax 전면 재설계 필요.
(현재 코드 최고 랭크 기준 280만 원 → 아래 스케일로 전면 조정)

| 직업 | 14일 기준 하한 | 14일 기준 상한 |
|------|--------------|--------------|
| 노숙자 / 백수 | 0원 | 50만 원 |
| 일용직 노동자 | 150만 원 | 400만 원 |
| 회사원 | 300만 원 | 600만 원 |
| 자영업자 | 500만 원 | 3,000만 원 |
| 도박사 | 0원 | 5억 원 (운 조건, 분산 극대화) |
| 사기꾼 | 1,000만 원 | 1억 원 |
| 마약상 | 3,000만 원 | 5억 원 |
| 투자자 | 1억 원 | 20억 원 |
| 보이스피싱 조직장 | 5억 원 | 50억 원 |
| 사이비 교주 | 10억 원 | 100억 원 |
| 대기업 오너 | 100억 원 | 1,000억 원 |
| 대통령 | 1억 원 | 3억 원 (월급 기준, 권력은 돈이 아님) |
| 어둠의 권력자 | 1조 원 | 상한 없음 |

### 탭별 표시 기준

| 탭 | 정렬 기준 | 표시 단위 |
|----|----------|----------|
| 재산 | 최종 돈 높은 순 | 조 / 억 / 만 원 |
| 범죄력 | 범죄도 높은 순 | 수치 + 대표 범죄 직업명 |
| 평판 | 평판 높은 순 | 수치 + 직업명 |
| 지능 | 지능 높은 순 | 수치 + 직업명 |

### 랭킹 원칙

- 탭마다 기준이 달라서 재산 1위가 평판 꼴찌일 수 있음
- 단위는 조 / 억 / 만 원 (한국식 표기)
- 다른 플레이어 직업명 + 수치 같이 표시
- 내 위치 하이라이트
- 배금도시 주제를 마지막에 각인 — 돈 많은 사람이 꼭 좋은 사람은 아님

### 돈 0 이벤트

게임오버 없음. 돈이 0이 되면 극빈 루트로 자연스럽게 연결.

```
돈이 0이 됐다
  → 배고픔에 쓰러진다
  → 병원에 실려 간다 (치료비 발생 → 빚 시작)
  → 빚진 상태로 다시 14일 루프
      → 더 절박해져서 범죄 선택지가 더 많이 열림
      → 아니면 악착같이 갚으면 평판 +
```

돈 0이 게임오버가 아니라 오히려 더 어두운 루트의 입구가 됨.

### 온라인 랭킹 구현

Firebase 또는 Supabase 연동 예정 (게임 완성 후 붙이기)
- 게임 종료 시 닉네임 + 직업명 + 최종 돈 → DB 저장
- 랭킹 화면에서 상위 N명 불러와서 표시
- 별도 백엔드 서버 없이 프론트엔드만으로 연결 가능

---

## 원칙

- 직업군은 선택의 결과지 목표가 아님. 자연스럽게 흘러들어가야 함
- 대통령 엔딩은 게임에서 제일 달성하기 어렵게 설계
- 어둠 엔딩은 의도하지 않아도 범죄도가 쌓이면 자연스럽게 흘러들어가도록
- 랭킹은 돈이 전부인 세상을 풍자하는 마지막 장치
