# 특수 이벤트 / 미니게임 설계

## 개념

일반 선택지와 다르게 별도 UI가 뜨는 이벤트.
특정 조건(스텟, 일차, 루트)을 만족할 때만 발동.

---

## 예술가 — 캔버스 이벤트

### 발동 조건
- 지능 일정 이상 + 창작 이벤트 루트 진입 시

### 진행 방식

```
캔버스 UI가 뜬다
  → 그림을 그린다 (간단한 드로잉 or 선택형 구성)
  → 완성된 작품을 어디에 올릴지 선택
      A. 길거리에서 판다   → 즉시 소액 수입, 평판 소폭 상승
      B. 인터넷에 올린다   → 하트 수에 따라 수입 결정 (시간 지연)
```

### 인터넷 업로드 결과

```
하트 0 ~ 50    → 수입 없음
하트 51 ~ 200  → 소액 (1만 ~ 5만 원)
하트 201 ~ 999 → 중간 (10만 ~ 50만 원)
하트 1000 이상 → 대박 (100만 원 이상, 평판 대폭 상승)
```

하트 수는 랜덤 + 평판 수치가 가중치로 작용.
평판 높을수록 하트 많이 받을 확률 증가.

### 온라인 연동 (향후)

인터넷에 올린 작품을 실제 온라인 커뮤니티에서 다른 플레이어들이 하트를 누를 수 있게 연동.
다른 플레이어가 하트 누르면 해당 플레이어의 게임 내 수입/평판에 실시간 반영.

```
플레이어 A가 작품 업로드
  → 커뮤니티 피드에 노출
  → 다른 플레이어들이 하트
  → A의 게임 내 수입 증가
```

## 도박 — 카드/주사위 이벤트

### 발동 조건
- 도박 장소 이벤트 발동 시 (특정 일차 or 범죄도 10 이상)

### 진행 방식

```
도박판 UI가 뜬다
  → 베팅 금액 입력
  → 카드 or 주사위 미니게임
  → 결과: 2배 획득 or 전액 손실
```

베팅 금액 제한 없음. 전재산 베팅 가능.
도박 중독 루트: 잃으면 다시 하고 싶어지는 선택지 등장 (범죄도 상승)

---

## 보이스피싱 — 전화 이벤트

### 발동 조건
- 범죄도 25 이상 + 보이스피싱 조직 가입 후

### 진행 방식

```
전화 UI가 뜬다
  → 대본이 주어짐
  → 플레이어가 선택지로 대화 진행
  → 상대방이 속으면 → 수입 발생, 범죄도 상승
  → 상대방이 끊으면 → 수입 없음
```

대화 선택지가 정교할수록 성공률 높아짐 (지능 수치 반영)

---

## 주식 / 코인 — 차트 이벤트

### 발동 조건
- 휴대폰 해금 후 주식/코인 앱 사용 시

### 진행 방식

```
간단한 차트 UI가 뜬다
  → 올라갈 것 같다 / 내려갈 것 같다 선택
  → 베팅
  → 결과 확인
```

지능 높을수록 힌트 제공 (차트 패턴 설명 텍스트 등장)
지능 낮으면 완전 랜덤

---

## 원칙

- 특수 이벤트는 별도 UI가 뜨는 것만 여기에 포함
- 일반 선택지 이벤트는 memory-system.md 참고
- 온라인 연동 기능은 게임 완성 후 Firebase/Supabase와 함께 붙이기
- 미니게임 UI는 현재 청소게임 구조(scene: "cleanup")를 확장해서 구현
