# 배금도시 경제 시스템

배금도시의 핵심은 돈이다. 모든 시스템은 돈을 중심으로 연결된다.
인플레이션을 이기는 싸움 — 가만히 있으면 가난해진다.

---

## 전체 구조

```
┌─────────────────────────────────────────┐
│              플레이어 재산               │
└──────┬──────────────────────┬───────────┘
       │ 수입                 │ 지출
  ─────┼─────            ─────┼─────
  알바  주식             식비  월세
  코인  사업             교통  아이템
  부동산 환율            세금  병원비
  범죄  이벤트보상        빚이자
       │
       ▼
  세계이벤트 (물가급등, 환율충격, 증시폭락)
```

---

## 1. 물가 시스템 (인플레이션)

매일 소폭 상승. 벌지 않으면 자동으로 가난해지는 구조.

| 항목 | 1일차 | 7일차 | 14일차 |
|------|-------|-------|--------|
| 식비 (하루) | 8,000원 | 10,000원 | 13,000원 |
| 교통비 | 1,500원 | 1,700원 | 2,000원 |
| 월세 | 300,000원 | — | 380,000원 (14일차 정산) |

- 물가 상승률은 `INFLATION_RATE` 상수로 조정
- 세계이벤트 발생 시 당일 물가 급등 가능
- 식비를 못 내면 → 체력 감소 → 결국 쓰러짐

---

## 2. 노동 수입

알바, 일용직, 사업 등 시간을 써서 버는 돈.

- 시급 기준으로 계산
- 소비한 시간만큼 하루가 줄어듦 (timeline 시스템 연동)
- 지능/평판이 높으면 더 좋은 일자리 해금

```
단순노동   시급 9,000원~12,000원
서비스직   시급 12,000원~18,000원
전문직     시급 30,000원 이상 (지능 필요)
사업       수익 변동 (리스크 있음)
범죄       수익 높음, 범죄도 상승
```

---

## 3. 주식

스마트폰 앱에서 매수/매도. 매일 가격 변동.

```
저위험   국채, 대기업주   수익 낮음 / 손실 낮음
중위험   중소기업주       수익 중간
고위험   테마주, 작전주   대박 or 폭락
```

- 지능 높으면 차트 힌트 제공
- 세계이벤트 발생 시 특정 종목 급등/급락
- 범죄도 높으면 작전세력 루트 해금 (주가조작 가담)

---

## 4. 코인 (비트코인)

선물 기준. 레버리지 기능 있음. 한방에 부자 or 파산.

- 하루에 2배 or 반토막 가능
- 돌발이벤트: 잘못 투자해서 전재산 날림
- 범죄도 높으면 코인 세탁 루트 해금

---

## 5. 환율

원화 대비 달러(USD) 환율이 매일 변동.

```
기준 환율: 1,300원/달러 (게임 시작 시)
```

**활용 방법:**
- 달러를 사두면 원화 약세 시 이익
- 세계이벤트 (전쟁, 외환위기 등) 발생 시 환율 급등
- 수입 물가에 영향: 환율 오르면 외제품/식재료 가격 상승

**돌발이벤트 예시:**
```
"미국이 금리를 올렸다. 달러가 1,500원을 돌파했다."
→ 달러 보유 플레이어: 수익
→ 수입품 구매 예정 플레이어: 손해
→ 주식 외국인 매도 → 증시 하락
```

- 환율 변동은 주식, 코인, 물가, 부동산에 연쇄 영향
- 지능 높으면 환율 예측 힌트 제공

---

## 6. 부동산

집 구매 시 배경 이미지 변경. progression 시스템 연동.

```
고시원 → 원룸 → 아파트 → 펜트하우스
```

- 전세사기 돌발이벤트 존재
- 시세차익 가능 (지능 + 평판 필요)
- 집 등급이 올라갈수록 다음 회차 시작 조건 개선

---

## 7. 돈 단위

현실 한국 단위 그대로.

```
원 → 만 → 십만 → 백만 → 천만 → 억 → 조
```

표시: 1,230,000원 → 123만 원 / 1,000,000,000원 → 10억 원

---

## 8. 위기 상황

**돈이 0이 됐을 때**
- 식비 없음 → 체력 감소 → 쓰러짐 → 병원 이송
- 병원비 발생 → 빚 시작
- 빚진 상태로 다음 루프 시작 (더 불리한 조건)

**빚 시스템**
- 매일 이자 발생
- 이자를 못 갚으면 사채업자 이벤트 발동
- 범죄도 높으면 빚을 범죄로 해결하는 선택지 등장

대출, 담보, 연체, 압류의 상세 구조는 별도 문서로 분리한다.

- 상세 규칙 문서: `docs/design/bank-loans-and-collateral.md`
- 현금/은행 분리와 출생 자산 패키지: `docs/design/wealth-and-cashflow.md`

---

## 9. 세계이벤트와 경제 연동

경제 시스템은 단독으로 돌아가지 않는다.
세계에서 일어나는 사건이 즉각 경제에 영향을 준다.

```
외환위기       →  환율 급등, 주식 폭락, 월세 상승
보이스피싱     →  가족 재산 손실 이벤트
전세사기       →  부동산 보증금 날림
코인 폭락장    →  코인 보유자 전재산 손실
자연재해       →  특정 아이템 가격 급등
정치 스캔들    →  증시 하락, 특정 종목 영향
```

---

## 10. 상세 소비 시스템

부동산, 자동차, 백화점, 옷, 휴대폰, 외모꾸미기, 운동하기처럼 `돈을 써서 스탯 / 인벤토리 / 소유 상태를 바꾸는 소비`는 별도 문서로 분리한다.

- 상세 규칙 문서: `docs/design/consumption.md`
