# 경제 시스템

## 핵심 구조

```
수입     알바, 주식, 코인, 사업, 범죄
지출     식비, 월세, 교통, 아이템 구매
변수     물가상승률 (매일 소폭 상승)
충격     세계이벤트 → 물가 급등
```

핵심 긴장감: 물가 상승률 > 내 수입 증가율이면 결국 파산.
단순히 돈을 모으는 게임이 아니라 인플레이션을 이기는 싸움.

---

## 물가 시스템

매일 물가가 소폭 상승. 가만히 있으면 가난해지는 구조.

| 항목 | 1일차 | 7일차 | 14일차 |
|------|-------|-------|--------|
| 라면 | 1,000원 | 1,200원 | 1,500원 |
| 교통비 | 1,500원 | 1,700원 | 2,000원 |
| 월세 | 300,000원 | 330,000원 | 380,000원 |

- 물가 상승률은 상수로 조정 가능 (기획 단계에서 밸런싱)
- 세계이벤트 발생 시 물가 급등 가능 (외환위기, 전쟁 등)
- 월세는 14일차 정산일에 한꺼번에 청구

---

## 주식

- 매일 가격 변동
- 스마트폰 앱에서 매수/매도
- 종목별 리스크 차이 존재
- 지능 수치 높으면 차트 힌트 제공

```
저위험  국채, 대기업주  →  수익 낮음, 손실 낮음
중위험  중소기업주      →  수익 중간
고위험  테마주, 작전주  →  대박 or 폭락
```

---

## 비트코인 / 코인

- 극단적 변동성 (하루에 2배 or 반토막)
- 한방에 부자 or 파산 가능
- 돌발이벤트: 코인 잘못 투자해서 전재산 날림
- 범죄도 높으면 작전세력과 연결되는 루트 해금

---

## 부동산

- 집 구매 시 배경 변경 (progression 시스템 연동)
- 전세사기 돌발이벤트 존재
- 시세차익으로 수익 가능 (지능 수치 필요)

```
고시원  →  원룸  →  아파트  →  펜트하우스
```

---

## 돈 단위

현실 한국 단위 그대로 사용.

```
원 → 만 → 십만 → 백만 → 천만 → 억 → 조
```

표시 예시: 1,230,000원 → 123만 원

---

## 돈이 0이 됐을 때

- 식비 없음 → 배고픔으로 쓰러짐 → 병원 이송
- 병원비 발생 → 빚 시작
- 빚진 상태로 다음 루프 시작 가능 (더 불리한 조건)
