# 인벤토리 시스템 설계

## 핵심 개념

플레이어가 소유한 아이템을 관리.
아이템은 루트/엔딩/선택지에 직접 영향을 미침.
특정 아이템 없으면 특정 선택지가 아예 안 보임.

---

## 아이템 카테고리

### 소지품 (팔 수 있음)

| 아이템 | 획득 방법 | 용도 |
|--------|----------|------|
| 중고 물건들 | 집 청소, 이벤트 | 당근마켓에서 판매 |
| 그림 작품 | 캔버스 이벤트 클리어 | 길거리 판매 or 인터넷 업로드 |
| 장물 | 절도 이벤트 | 당근마켓 or 암시장 판매 (범죄도 상승) |

### 장비 (효과 지속)

| 아이템 | 획득 방법 | 효과 |
|--------|----------|------|
| 스마트폰 | 3일차 부모님에게 받음 | 앱 사용 가능, 각종 이벤트 해금 |
| 노트북 | 구매 or 이벤트 | 주식/코인 투자 효율 상승, 온라인 활동 가능 |
| 운동복 | 구매 | 에너지 회복량 증가 |
| 정장 | 구매 | 전문직 알바 지원 가능, 평판 상승 |
| 흉기 | 범죄 이벤트 | 범죄 선택지 추가 해금 (소지만 해도 범죄도 상승) |
| 화구 (물감/붓) | 구매 | 캔버스 이벤트 발동 조건 |

### 문서 / 자격증

| 아이템 | 획득 방법 | 효과 |
|--------|----------|------|
| 사업자등록증 | 창업 이벤트 클리어 | 자영업자 루트 진입 가능 |
| 운전면허증 | 특정 이벤트 | 배달 알바 가능 |
| 대학 합격증 | 지능 높음 + 이벤트 | 전문직 루트 진입 조건 |
| 가짜 신분증 | 범죄도 25 이상 + 이벤트 | 사기 선택지 확장 |

### 소모품

| 아이템 | 획득 방법 | 효과 |
|--------|----------|------|
| 편의점 도시락 | 구매 (3,000원) | 에너지 소폭 회복 |
| 에너지 드링크 | 구매 (2,000원) | 에너지 즉시 회복, 체력 소폭 감소 |
| 약 | 병원 or 구매 | 체력 회복 |

## UI 구조

```
가방 버튼 누르면 → 인벤토리 팝업

[소지품]  [장비]  [문서]  [소모품]

아이템 클릭 시
  → 사용 / 팔기 / 버리기 선택
```

---

## 원칙

- 인벤토리 슬롯 제한 있음 (초반 8칸, 확장 가능)
- 특정 아이템 소지 자체가 이벤트 트리거 조건
- 흉기 소지 → 경찰 이벤트 발동 가능성 상승
- 장물 소지 → 집에 경찰이 오는 이벤트 발동 가능성 상승
- 당근마켓은 소지품만 판매 가능, 장물은 암시장 전용
- 아이템은 돈으로 대부분 구매 가능하지만 일부는 이벤트로만 획득
