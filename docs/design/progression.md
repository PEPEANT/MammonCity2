# 메타 진행 시스템 (로그인 + 집 + 다음 회차)

## 14일의 이유

```
부모님: "14일 안에 월세 낼 돈 만들어. 못 하면 나가."
```

14일은 단순한 게임 제한이 아니라 서사적 데드라인.
월세를 내느냐 못 내느냐가 1회차 엔딩의 분기점이 됨.

| 결과 | 연결 엔딩 |
|------|----------|
| 월세 못 냄 | 노숙자 엔딩 or 극빈 루트 |
| 월세만 겨우 냄 | 부모님 집 유지, 다음 회차도 동일 시작 |
| 월세 + 여유 있음 | 원룸 이사 가능, 다음 회차 시작점 상승 |
| 큰 돈 벌음 | 더 높은 등급 집 구매, 빠른 진행 |

---

## 핵심 개념

14일이 끝나도 게임이 끝나지 않는다.
로그인한 플레이어는 전 회차 결과가 다음 회차 시작 조건에 영향을 미친다.
집을 사면 다음 회차의 출발점이 바뀐다.

---

## 회차 구조

```
1회차
  부모님 집에서 시작 (흙수저 기본값)
  14일 플레이
  결과에 따라 집 구매 가능

2회차 이후
  구매한 집에서 시작
  집 등급에 따라 시작 조건이 달라짐
  이전 회차 기억 일부 유지 (스텟, 인맥, 아이템)
```

---

## 집 등급표

| 등급 | 집 종류 | 구매 조건 | 다음 회차 시작 보너스 |
|------|--------|----------|-------------------|
| 0    | 부모님 집 | 기본값 | 없음 (흙수저 시작) |
| 1    | 고시원 | 50만 원 이상 | 시작 돈 소폭 증가 |
| 2    | 원룸 | 500만 원 이상 | 시작 돈 증가, 에너지 회복량 증가 |
| 3    | 오피스텔 | 3,000만 원 이상 | 시작 스텟 보너스, 일부 알바 처음부터 해금 |
| 4    | 아파트 | 1억 원 이상 | 평판 시작값 증가, 전문직 루트 빠른 접근 |
| 5    | 펜트하우스 | 10억 원 이상 | 대부분 루트 초반부터 열림 |
| 6    | 비밀 아지트 | 범죄도 90 이상 | 범죄 선택지 처음부터 해금, 경찰 이벤트 지연 |

---

## 로그인 시스템

로그인하면 플레이어 데이터가 서버에 저장됨.

```
저장되는 것
  - 현재 보유 집 등급
  - 누적 최고 재산
  - 달성한 엔딩 목록
  - 랭킹 순위

저장 안 되는 것 (회차마다 초기화)
  - 돈 (집 구매 후 잔액은 일부만 이월)
  - 에너지 / 체력
  - 그날그날 이벤트 상태
```

---

## 회차 간 이월 시스템

```
전 회차에서 다음 회차로 넘어가는 것

집 등급         → 유지 (그 집에서 시작)
달성 엔딩       → 기록 유지 (도감 형태로 모을 수 있음)
인맥 일부       → 특정 NPC와의 관계가 다음 회차에 남아있음
```

---

## 집이 다른 분기 시작점이 되는 구조

```
부모님 집에서 시작  →  흙수저 서사 그대로
원룸에서 시작       →  "드디어 독립했다" 서사로 시작
아파트에서 시작     →  "이미 성공한 사람" 서사로 시작
비밀 아지트 시작    →  "이미 범죄 조직과 연결된" 서사로 시작
```

집에 따라 1일차 텍스트, 배경, 초기 선택지가 전부 달라짐.

---

## 원칙

- 로그인 없이도 플레이 가능 (게스트 모드 = 매번 부모님 집 시작)
- 로그인하면 집이 누적되고 더 다양한 시작 조건 잠금 해제
- 집은 한 번 사면 영구 보유
- 14일이 끝이 아니라 새로운 14일의 시작점을 바꾸는 구조
- Firebase / Supabase 로그인 연동 (랭킹 시스템과 동일한 DB)
