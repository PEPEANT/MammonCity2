# 메모리 시스템 설계 아이디어

## 핵심 개념

1일차 선택이 이후 모든 일차에 영향을 미친다.
같은 이벤트라도 1일차 기억에 따라 버튼명/대사/발동 이벤트가 달라진다.

---

## 1일차 분기점

주인공은 남성. 1일차에 반드시 하나를 선택하게 됨.

| 선택 | 결과 | 기본 경로 |
|------|------|----------|
| 여자한테 차인다 | 상처: "차임" 생성 | 결핍 루트 |
| 여자와 잘 된다 | 상처 없음, 성향: "안정" | 평범 / 사업가 루트 유리 |
| 여자를 아예 만나지 않는다 | 상처 없음, 성향: "무관심" | 지능/돈 집중형 루트 유리 |

차이지 않은 경우에도 이후 이벤트는 발동되지만 버튼명/대사가 결핍 루트와 다르게 나옴.

---

## 파생 경로

```
상처 (여자한테 차임)
  → 연애/외모 집착형 (성형, 데이트 스킬)
      → 실패 시 → 범죄자 or 과시형 남자
  → 사업/창업 루트 (상처 상관없이 동일하게 존재)
```

---

## 구현 방식 (A안)

```js
state.기억 = {
  상처: "차임",     // 1일차 선택으로 결정 ("차임" / "안정" / "무관심")
  성향: "결핍",     // 상처에서 파생되는 성향
  범죄도: 0,        // 숨김. 나쁜 선택을 할수록 쌓임. 선택지 잠금 해제 조건
  지능: 0,          // 도서관 공부로 상승. 고급 루트 조건
  외모: 0,          // 돈 씀 (성형/옷), 특정 이벤트로 상승. 하락 조건: 교통사고/폭행
  평판: 0           // 선택에 따라 양방향. 플레이어에게 보임
}
```

### 레이어 1 — 버튼명/대사 변경
같은 이벤트라도 기억에 따라 텍스트가 달라짐

```
기본:  "오늘 일을 구한다"
차임:  "돈을 벌어야 그녀한테 보여줄 수 있다"
```

### 레이어 2 — 기억 전용 이벤트 (특정 일차)
14일 전부 다르게 만들 필요 없음. 5~6개 일차에만 꽂기

```
5일차:  길에서 커플을 봤다        (상처: 차임 일 때만 발동)
14일차: 성형외과 공고가 뜬다
20일차: 선택 — 과시형 소비 or 자기개발
```

### 레이어 3 — 엔딩 분기
상처 + 성향 + 범죄도 + 지능 조합으로 최종 엔딩 결정

| 범죄도 | 지능  | 엔딩 유형 |
|--------|-------|----------|
| 낮음   | 낮음  | 평범하게 살다 끝 |
| 낮음   | 높음  | 합법적 사업가 / 정치인 |
| 높음   | 낮음  | 충동적 범죄자 / 사이비 교주 |
| 높음   | 높음  | 정치 쿠데타 / 조직적 사기 / 권력자 |

## 세계 이벤트 시스템

주인공이 의도하지 않아도 외부에서 강제로 선택을 만드는 이벤트.
플레이어가 주도하는 게 아니라 세상이 먼저 움직이는 구조.

### 개념

```
트리거: 특정 일차 or 상태 조건
발동:   선택지가 강제로 등장
선택지: 성향/범죄도 수치에 따라 열리는 옵션이 다름
```

### 예시

| 이벤트 | 기본 선택지 | 조건부 선택지 |
|--------|------------|--------------|
| 부모님이 창업을 하신다 | 말린다 (혼이 난다) | 돈을 훔친다 (범죄도 일정 이상일 때만 노출) |
| 동네에 사건사고 발생 | 모른 척한다 | 가담한다 (범죄도 조건) |

### 범죄도 단계표

범죄도가 특정 구간을 넘으면 선택지가 순차적으로 해금됨

| 수치  | 해금 내용 |
|-------|----------|
| 0     | 기본 선택지만 보임 |
| 10 이상 | 소액 절도, 사기 선택지 노출 |
| 25 이상 | 폭행, 협박 선택지 노출 |
| 50 이상 | 살인 선택지 노출 |
| 40 이상 | 마약 거래 선택지 노출 |
| 70 이상 | 사이비 종교 창설 루트 해금 |
| 90 이상 | 쿠데타 루트 해금 |
| 90 이상 + 지능 80 이상 | 정치인 쿠데타 루트 해금 (범죄도 + 지능 동시 조건) |

### 원칙

- 세계 이벤트는 플레이어가 선택하지 않아도 발동됨
- 범죄도가 낮으면 나쁜 선택지 자체가 안 보임 (존재를 모름)
- 세계 이벤트도 레이어 2처럼 특정 일차에만 꽂기

## 스텟 설계

### 항상 보이는 스텟 (생존용)
```
에너지 / 체력 / 돈 / 시간
```
매일 쓰는 수치라 항상 화면에 표시

### 숨겨진 스텟 (성장/분기용)
버튼 하나 누르면 슬라이드 or 팝업으로 표시

```
지능    ██░░░░░░  14   ← 도서관 공부로 오름
외모    ███░░░░░  19   ← 돈 씀 (성형, 옷), 특정 이벤트
평판    █████░░░  38   ← 선택에 따라 양방향
```

### 범죄도는 플레이어에게 숨김
플레이어 화면에 안 보이고 내부에서만 계산.
어느 날 갑자기 새로운 선택지가 뜨면서 "나 이렇게까지 됐나" 하는 느낌을 줌.
수치가 보이면 플레이어가 계산하면서 플레이하게 돼서 몰입이 깨짐.

### 스텟 조합 → 엔딩

| 외모  | 평판  | 엔딩 유형 |
|-------|-------|----------|
| 높음  | 높음  | 카리스마형 사기꾼 / 정치인 |
| 높음  | 낮음  | 껍데기만 남은 과시형 |
| 낮음  | 높음  | 실력으로 인정받는 사업가 |
| 낮음  | 낮음  | 아무도 모르는 조용한 범죄자 |

### 스텟 올리는 방법

| 스텟  | 방법 |
|-------|------|
| 지능  | 도서관 방문 → 공부 (에너지 소모) |
| 외모  | 돈 씀 (성형, 옷 구매), 특정 이벤트 |
| 평판  | 선택에 따라 오르거나 내림 |
| 범죄도 | 나쁜 선택을 할수록 쌓임 (숨김) |

### 도서관 씬

```
밖에 나왔다
  → 구인게시판  (돈 버는 루트)
  → 도서관      (지능 오르는 루트, 에너지 소모)
  → 집으로
```

핵심 긴장감: 도서관 가면 지능은 오르는데 그날 일을 못 하거나 에너지가 줄어듦.
돈이냐 지능이냐 매일 선택해야 하는 구조.

## 돌발 이벤트 시스템

게임 전체 방향은 "배금도시 = 사기의 도시". 흙수저 주인공이라 세상이 계속 덮쳐오는 구조.
플레이어가 준비 안 된 상태에서 갑자기 발동. 막으면 지키고 못 막으면 돈/스텟 손실.

### 가족 관련

| 이벤트 | 결과 |
|--------|------|
| 부모님이 보이스피싱 사기를 당한다 | 막으면 돈 유지 / 못 막으면 가족 돈 증발 |
| 부모님이 창업했다가 망한다 | 빚 일부 떠넘겨짐 |
| 사이비 종교가 부모님한테 접근한다 | 초반부터 발동 가능. 범죄도 높으면 역으로 이용 가능 |

### 금융 관련

| 이벤트 | 결과 |
|--------|------|
| 코인 투자 실수 | 한 순간에 보유 돈 일부 or 전부 증발 |
| 전세 사기를 당한다 | 큰 돈 날림. 지능 높으면 사전에 눈치채고 막을 수 있음 |
| 지인이 투자 사기를 권유한다 | 거절 or 참여 (참여 시 대박 or 쪽박) |

### 사고/재해

| 이벤트 | 결과 |
|--------|------|
| 교통사고를 당한다 | 체력/에너지 큰 손실. 며칠 활동 불가 |
| 자연재해 발생 | 특정 장소 이용 불가. 돌발 선택 발생 |
| 묻지마 폭행을 당한다 | 체력 손실. 범죄도 높으면 역으로 가담자가 될 수도 |

### 원칙

- 흙수저라서 당연히 이런 일이 일어난다는 전제
- 막을 수 있는 이벤트는 지능/평판/돈 조건이 있음
- 못 막으면 페널티, 막으면 오히려 평판 or 지능 보상
- 범죄도 높으면 피해자가 아니라 가담자/이용자가 되는 선택지 등장
- 휴대폰 해금 이후에만 발동되는 이벤트 별도 존재 (코인, 보이스피싱 등)

### 사이비 종교 연결 흐름

초반엔 피해자로 만나고, 범죄도가 쌓이면 나중엔 교주가 되는 구조.
같은 사이비가 처음과 끝에 다른 방식으로 등장.

```
초반 (낮은 범죄도)
  → 사이비가 부모님한테 접근한다
  → 막는다 / 모른 척한다

중반 (범죄도 25 이상)
  → 사이비 조직원이 주인공한테 직접 접근한다
  → 거절한다 / 가입한다

후반 (범죄도 70 이상)
  → 사이비 종교 창설 루트 해금
  → 초반에 당할 뻔했던 그 수법을 직접 쓰는 교주가 됨
```

피해자 → 방관자 → 가담자 → 교주 로 자연스럽게 이어지는 흐름.

### 보이스피싱 연결 흐름

사이비와 동일한 구조. 처음엔 피해자 가족을 보호하다가 결국 가담자가 되는 흐름.

```
초반 (낮은 범죄도)
  → 부모님이 보이스피싱 사기를 당하고 있다
  → 막는다 / 못 막는다 (돈 손실)

중반 (범죄도 25 이상)
  → 보이스피싱 조직원이 아르바이트를 제안한다
  → 거절한다 / 가입한다 (단기 고수익)

후반 (범죄도 50 이상)
  → 조직 내에서 역할이 커짐
  → 탈출한다 / 계속한다 / 직접 조직을 차린다

```

피해자 가족을 둔 주인공이 결국 같은 조직을 운영하게 되는 흐름.
보이스피싱과 사이비는 서로 다른 루트지만 범죄도 수치는 공유해서 쌓임.

---

## 원칙

- 기본 루프 (일 선택 → 이벤트 → 결과) 는 동일하게 유지
- 기억은 연출과 분기에만 영향
- 사업/창업 루트는 어떤 상처든 접근 가능
- 세계 이벤트는 플레이어를 능동적으로 만드는 장치가 아니라 세상이 주인공을 시험하는 장치
- 범죄도는 플레이어에게 숨겨서 몰입감 유지
