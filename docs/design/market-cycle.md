# 12턴 시장 사이클
1턴을 1개월로 보고, 12턴 전체를 하나의 연간 시장 캘린더로 묶어 모든 자산과 뉴스가 같은 방향성을 공유하게 만든다.

## 목적

- 시장을 `완전 랜덤`이 아니라 `읽을 수 있는 흐름`으로 바꾼다.
- 플레이어가 현재 국면과 다음 국면을 한눈에 보게 만든다.
- 주식, 코인, 금, 환율, 뉴스, Diggle 검색 결과가 같은 시장 소스에서 움직이게 만든다.
- 호스팅 화면에서는 12턴 전체 흐름과 현재 위치를 한 번에 보여준다.

## 핵심 결정

- `1턴 = 1개월`
- `1 ~ 12턴 = 하나의 연간 시장 사이클`
- 시장 상태는 분리된 랜덤값이 아니라 `공통 캘린더`에서 읽는다.
- 코인은 주식보다 빠르게 반응하는 `선행 자산`으로 둔다.
- 뉴스는 랜덤 생성보다 `턴별 미리 작성된 뉴스 풀`을 우선한다.
- 플레이어는 항상 `현재 턴`, `현재 국면`, `다음 변동성 힌트`를 볼 수 있어야 한다.

## 한눈에 보는 구조

```text
12턴 시장 캘린더
  -> 현재 턴 / 현재 월
  -> 현재 국면
  -> 공포탐욕 수치
  -> 주식 반응
  -> 코인 반응
  -> 금 / 환율 반응
  -> 이번 턴 핵심 뉴스
  -> 다음 턴 예고 뉴스
```

## 자산 반응 원칙

### 주식

- 거시 국면을 가장 정직하게 따라간다.
- 코인보다 한 박자 늦다.
- 호황 구간은 길게 먹지만, 급반등과 급락은 코인보다 둔하다.

### 코인

- `선반영 + 과장 반응` 자산으로 둔다.
- 회복도 먼저, 과열도 먼저, 붕괴도 먼저 온다.
- 주식보다 기대감과 공포를 크게 먹는다.
- 다만 항상 정답이 되지 않게 `가짜 반등`과 `오버슈트`를 일부 허용한다.

### 금

- 공포 구간에서 강해진다.
- 후퇴기, 침체기, 위험회피 장세에서 수요가 붙는다.
- 호황기에는 재미가 덜하지만 방어 자산 역할을 한다.

### 환율

- 위험회피와 연동해 상승하기 쉽다.
- 침체기 / 공포기에는 원달러가 튀고, 회복기 / 호황기에는 안정되거나 내려간다.

## 선행/후행 규칙

### 코인 선행 규칙

- 코인은 주식보다 `1턴 빠르게` 국면 변화를 반영한다.
- 회복 전환 직전 턴에 먼저 반응한다.
- 과열 정점도 먼저 찍고 먼저 무너진다.

예시:

- 턴 3: 주식은 아직 회복 확인 전, 코인은 이미 기대감 반등
- 턴 7: 주식은 아직 호황 후반, 코인은 정점권 과열
- 턴 8: 주식은 흔들리기 시작, 코인은 먼저 하락 전환

### 주식 후행 규칙

- 주식은 코인보다 `1턴 늦게` 시장 기대를 가격에 반영한다.
- 대신 코인보다 지속성이 길다.
- 코인이 튄 뒤 주식이 따라오거나, 코인이 무너진 뒤 주식이 늦게 조정받는 흐름을 만든다.

## 뉴스 설계 원칙

- 뉴스는 `턴별 사전 작성`이 기본이다.
- 각 턴마다 최소 `거시 1개 + 자산군 뉴스 2개`를 준비한다.
- 뉴스는 시장과 따로 놀면 안 된다.
- 같은 국면이라도 `기본 뉴스`와 `변형 뉴스`를 섞어 반복 피로를 줄인다.

### 권장 뉴스 슬롯

| 슬롯 | 설명 |
| --- | --- |
| 거시 뉴스 | 금리, 유동성, 경기, 소비, 수출, 정책 |
| 주식 뉴스 | 실적, 반도체, AI, 수급, 외국인 매매 |
| 코인 뉴스 | ETF, 규제, 해킹, 고래, 밈코인 과열 |
| 안전자산 뉴스 | 금값, 환율, 위험회피 |
| 지역/생활 뉴스 | 배금시 체감 경기, 생활물가, 체감 취업시장 |

## 턴별 뉴스 초안

### 1턴

- 거시: 긴축 우려 재점화, 소비 둔화 전망
- 주식: 대형주 실적 경고, 지수 방어 실패
- 코인: 레버리지 청산, 단기 반등 실패
- 안전자산: 금값 반등, 환율 상승

### 2턴

- 거시: 기업 투자 위축, 신용 경색 우려
- 주식: 중소형주 낙폭 확대
- 코인: 거래량 죽은 바닥 구간, 변동성만 큼
- 안전자산: 현금/달러 선호 확대

### 3턴

- 거시: 금리 인하 기대 등장
- 주식: 바닥권 매수세 유입
- 코인: 기관 자금 유입 루머, 선반영 상승
- 안전자산: 금/환율 상승 둔화

### 4턴

- 거시: 회복 기대, 유동성 완화 시그널
- 주식: 성장주 회복
- 코인: 대장 코인 급반등
- 안전자산: 차익실현 구간

### 5턴

- 거시: 수출 회복, 경기 민감주 반등
- 주식: 반도체/AI 랠리
- 코인: 알트코인 순환매
- 안전자산: 관심 감소

### 6턴

- 거시: 낙관 확대, 실적 기대감 강화
- 주식: 강한 상승 추세
- 코인: 밈코인 포함 전반 급등
- 안전자산: 약세 유지

### 7턴

- 거시: 아직 좋지만 과열 경고
- 주식: 고평가 논쟁
- 코인: 거래대금 폭증, FOMO, 고래 추격 매수
- 안전자산: 최저 관심

### 8턴

- 거시: 차익실현과 부담 확대
- 주식: 상승 피로 누적
- 코인: 먼저 급락, 상투 논쟁
- 안전자산: 다시 매수 유입

### 9턴

- 거시: 경기 둔화 재부각
- 주식: 실적 실망, 조정 확대
- 코인: 데드캣 바운스와 재차 하락
- 안전자산: 강세 지속

### 10턴

- 거시: 위험회피 극대화
- 주식: 투매, 하방 이탈
- 코인: 해킹/청산/규제 악재 집중
- 안전자산: 금값 상승, 환율 급등

### 11턴

- 거시: 완화 기대, 바닥 논쟁
- 주식: 낙폭 축소
- 코인: 먼저 반등 시도
- 안전자산: 강세 둔화

### 12턴

- 거시: 연말 기대, 다음 해 회복 시선
- 주식: 선택적 반등
- 코인: 기술적 반등 확대
- 안전자산: 안정

## 대시보드 요구사항

호스팅에서 보는 통합 화면은 아래 정보를 한 번에 보여줘야 한다.

1. 현재 턴과 월
2. 12턴 전체 시장 캘린더
3. 현재 마커와 다음 턴 예고
4. 공포탐욕 수치와 최근 흐름
5. 주식 / 코인 / 금 / 환율 방향 화살표

### 화면 메모

- `현재 위치`는 12칸 리본이나 곡선 위 마커로 표시
- 현재 턴은 진하게, 지난 턴은 흐리게, 다음 턴은 예고 색으로 표기
- 코인과 주식은 같은 차트에 두지 말고 `선행/후행` 설명을 따로 붙인다
- 뉴스는 상세 기사보다 `헤드라인 + 한 줄 해설` 위주로 본다

## 계산 원칙

- 큰 방향은 `시장 국면`이 정한다.
- 자산별 차이는 `자산 성향`이 만든다.
- 뉴스는 시장 상태를 설명하거나 약하게 증폭한다.
- 마지막에만 소량 랜덤을 얹는다.

권장 비중:

- 시장 국면: 60%
- 자산 성향: 25%
- 이벤트/뉴스: 10%
- 순수 랜덤: 5%

## 구현 우선순위

1. 12턴 시장 캘린더 데이터부터 고정한다.
2. 턴별 뉴스 팩을 미리 작성한다.
3. 주식 / 코인 / 금 / 환율이 같은 캘린더를 읽게 바꾼다.
4. 호스팅용 통합 대시보드를 만든다.
5. Diggle / 뉴스 앱이 같은 뉴스 팩을 읽게 연결한다.

## 리스크

- 12턴 흐름이 너무 고정되면 외워서 풀게 된다.
- 코인이 너무 자주 정답이 되면 다른 자산이 의미를 잃는다.
- 뉴스 수가 부족하면 반복감이 빠르게 온다.
- 공포탐욕과 실제 자산 방향이 따로 놀면 신뢰가 깨진다.
